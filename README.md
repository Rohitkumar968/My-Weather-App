☁️ Advanced Weather App

A modern and responsive Weather App built using HTML, CSS, and JavaScript with OpenWeather API integration.

## 🚀 Features

- Search weather by city name
- Current location weather using Geolocation
- 5-Day Weather Forecast
- Temperature in Celsius
- Humidity, Wind Speed, Pressure
- Feels Like Temperature
- Dark Mode Toggle
- Loading Spinner
- LocalStorage Support
- Responsive Design
- Clean UI with Glassmorphism Effect

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API
